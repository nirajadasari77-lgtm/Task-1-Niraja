# Rule-Based AI Chatbot

## Project Overview
This project is a simple Rule-Based AI Chatbot developed using Python. The chatbot responds to user questions based on predefined rules and keywords. It demonstrates the basic concepts of artificial intelligence, conditional statements, and user interaction.

## Features
- Greets users professionally
- Responds to common questions
- Provides predefined answers
- Simple and easy-to-understand Python code
- Beginner-friendly AI project

## Technologies Used
- Python
- Visual Studio Code (VS Code)

## How It Works
The chatbot takes input from the user and checks it against predefined conditions. Based on the matching keyword or phrase, it returns an appropriate response.

## Sample Questions
- Hello
- How are you?
- What is AI?
- What is Machine Learning?
- What can you do?
- Bye

## How to Run the Project
1. Install Python.
2. Open the project in VS Code.
3. Run the chatbot file:

```bash
python chatbot.py
```

4. Start chatting with the bot.
